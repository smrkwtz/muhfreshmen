# This directory holds manually exported Barttorvik T-Rank CSVs.
# One file per tournament year, named barttorvik_YYYY.csv.
#
# HOW TO EXPORT:
#   1. Go to https://barttorvik.com/trank.php?year=YYYY
#   2. Set the end date filter to the day before the tournament starts
#      (see src/constants.py → TOURNAMENT_START_DATES for each year's date)
#   3. Click the Export button (top-right of the table)
#   4. Save the downloaded file here as barttorvik_YYYY.csv
#

#   2010: 20100318    2011: 20110315    2012: 20120313
